# <a name="cast-from-type-lttypename1gt-to-type-lttypename2gt-is-not-valid"></a>Türden atama '&lt;typename1&gt;'türü için '&lt;typename2&gt;' geçerli değil
Bir değişken bir türden diğerine dönüştürme girişimi geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Türleri dönüştürme ilgili denetleyin ve anlamlı bir türe dönüştürmeye çalışıyorsunuz. emin olun.
