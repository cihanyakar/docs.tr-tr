# <a name="event-lteventnamegt-event-specified-by-the-defaultevent-attribute-is-not-a-publicly-accessible-event-for-this-class"></a>Olay '&lt;eventname&gt;' 'DefaultEvent' özniteliği tarafından belirtilen olay bu sınıf için ortak olarak erişilebilen bir olay değil
<xref:System.ComponentModel.DefaultEventAttribute> Özniteliği öznitelik uygulandığı sınıfta ortak olarak erişilebilen olayın adını belirtmeniz gerekir.  
  
 **Hata Kimliği:** BC30770  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Genel olarak erişilebilen bir olay sınıfını tanımlar emin olun.  
  
2.  Sınıf içinde olay adı tarafından belirtilen adla eşleştiğinden emin olun <xref:System.ComponentModel.DefaultEventAttribute> özniteliği.
