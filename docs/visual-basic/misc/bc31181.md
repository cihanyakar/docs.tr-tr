# <a name="attribute-value-is-not-valid-expecting-value"></a>Öznitelik değeri geçerli değil; 'value' bekleniyor
Bir XML belgesi değişmez değer özniteliği için geçersiz değer belirtildi. Bir XML belgesi değişmez değeri geçerli bir öznitelik değerleri içinde XML 1.0 belirtimi belirtilir.  
  
 **Hata Kimliği:** BC31181  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Öznitelik değeri XML 1.0 belirtimi belirtildiği gibi geçerli bir değere güncelleştirin.
