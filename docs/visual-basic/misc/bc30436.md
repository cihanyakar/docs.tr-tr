# <a name="members-in-a-module-cannot-be-declared-ltspecifiergt"></a>Modül içindeki üyeleri bildirilemez '&lt;belirticisi&gt;'
İçinde bir üyesi üzerinde geçerli değil bir tanımlayıcı kullandığınız bir `Module` deyimi. Modüller asla oluşturulamaz, devralmayı desteklemeyen ve arabirimler uygulanamaz.  
  
 **Hata Kimliği:** BC30436  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcısını kaldırın.
