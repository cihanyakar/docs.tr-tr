# <a name="events-declared-with-an-as-clause-must-have-a-delegate-type"></a>Bir 'As' yan tümcesi ile bildirilen olaylarda bir temsilci türünde olmalıdır
Bir temsilci olmayan bir tür belirtildi.  
  
 **Hata Kimliği:** BC31044  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `As` yan tümcesi.  
  
2.  Bir temsilci ile belirtin `As` yan tümcesi.
