# <a name="events-in-interfaces-cannot-be-declared-ltimplementsgt"></a>Arabirimlerdeki olaylar bildirilemez '&lt;uygulayan&gt;'
Arabirimlerde bildirilen olaylarda olaylarını diğer arabirimler uygulayamaz.  
  
 **Hata Kimliği:** BC30688  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `Implements` deyimi.  
  
2.  Bir sınıf veya yapı içinde olay uygulayın.
