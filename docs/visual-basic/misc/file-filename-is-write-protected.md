# <a name="file-ltfilenamegt-is-write-protected"></a>Dosya '&lt;filename&gt;' yazma korumalı
Yazılacak çalıştığınız dosya yazma korumalı ise ve bu nedenle değiştirilemez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosya okuma/yazma veya farklı bir dosyaya yazmak için izinleri değiştirin.
