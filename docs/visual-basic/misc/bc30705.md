# <a name="referenced-object-ltobjectnamegt-has-a-value-of-nothing"></a>Başvurulan nesne '&lt;objectname&gt;' değeri 'Nothing'
Başvurulan nesne ayarlanmamışsa veya değerine ayarlanırsa `Nothing`.  
  
 **Hata Kimliği:** BC30705  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Nesnenin adını doğru yazdığınızdan ve nesne ayarlandığını doğrulayın.
