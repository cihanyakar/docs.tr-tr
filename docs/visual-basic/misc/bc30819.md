# <a name="ltmathfunction1gt-is-not-declared"></a>'&lt;mathfunction1&gt;' bildirimi yapılmadı
'\<mathfunction1 >' değil olarak bildirilir. İşlev System.Math sınıfa taşındı ve artık çağrılır '\<mathfunction2 >'.  
  
 Önceki sürümlerde Visual Basic iç çeşitli işlevler taşındı <xref:System.Math?displayProperty=nameWithType> ad alanı. Bu işlevleri tüm programlama dilleri için daha genel olarak kullanılabilir hale getirir.  
  
 **Hata Kimliği:** BC30819  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bildirilmiş yöntemleri kullanın <xref:System.Math?displayProperty=nameWithType>.
