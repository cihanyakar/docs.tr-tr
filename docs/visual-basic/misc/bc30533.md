# <a name="conversion-from-double-to-date-requires-calling-the-datefromoadate"></a>'Date.FromOADate' çağrılırken 'Date' 'Double' dönüştürme gerektiriyor
Cast istediniz bir `Date` değerini bir `Double` kullanmadan yapılamaz değeri <xref:System.DateTime.FromOADate%2A?displayProperty=nameWithType> yöntemi.  
  
 **Hata Kimliği:** BC30533  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım <xref:System.DateTime.FromOADate%2A> değerini dönüştürmek için yöntemi.
