# <a name="cannot-convert-start-value-of-type-lttype1gt-limit-value-of-lttype2gt-and-step-value-of-lttype3gt-to-a-common-type"></a>Başlangıç değeri türü dönüştürülemiyor '&lt;type1&gt;', değeri sınırlamak '&lt;type2&gt;'ve adım değerini'&lt;type3&gt;' ortak bir türe
Belirtilen `For...Next` denetim değerleri farklı türlere sahip ve ortak bir türe dönüştürülemediğinden bu nedenle geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bir veya daha fazla değer türünü, tümü ortak bir türe dönüştürülebilir şekilde değiştirin.
