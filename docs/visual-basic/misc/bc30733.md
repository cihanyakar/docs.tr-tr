# <a name="ltmembernamegt-is-already-declared-by-ltprocedurenamegt-which-was-generated-for-this-ltcontainernamegt"></a>'&lt;membername&gt;'zaten tarafından bildirilen'&lt;procedurename&gt;', oluşturulduğu için bu &lt;containername&gt;
Comom dil çalışma zamanı tarafından bildirilen bir üye ile aynı ada sahip bir üye bildirildi.  
  
 **Hata Kimliği:** BC30733  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Çakışma kaldırmak için üyenin adını değiştirin.
