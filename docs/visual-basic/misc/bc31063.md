# <a name="set-method-cannot-have-more-than-one-parameter"></a>'Set' yöntemi, birden fazla parametresine sahip olamaz
Birden fazla parametre bir özelliği için tanımlanan `Set` yordamları.  
  
 **Hata Kimliği:** BC31063  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Herhangi bir ek parametre özelliğinden kaldırılacak `Set` yordamları.
