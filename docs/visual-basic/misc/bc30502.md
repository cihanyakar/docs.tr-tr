# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-property-declaration"></a>İle 'Shared' birleştirilemez '&lt;belirticisi&gt;' özellik bildiriminde
Birleştirilecek çalıştınız `Shared` değiştiricisi ile bu tür bir birleşimini özellik bildiriminde geçerli değil bir tanımlayıcı.  
  
 **Hata Kimliği:** BC30502  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcısını kaldırın.
