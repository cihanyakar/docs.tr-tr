# <a name="argument-access-is-not-valid-valid-values-for-append-mode-are-openaccesswrite-and-openaccessdefault"></a>'Erişimi' bağımsız değişkeni geçerli değil. Ekleme modu için geçerli değerler şunlardır: 'OpenAccess.Write' ve 'OpenAccess.Default'
Ekleme modunda erişim değeri geçersiz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Değer ya da değiştirme `OpenAccess.Write` veya `OpenAccess.Default`.
