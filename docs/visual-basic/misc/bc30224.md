# <a name="is-expected"></a>Is bekleniyor
A `TypeOf` işleci gerçekleşir olmadan bir `Is` yan tümcesi.  
  
 **Hata Kimliği:** BC30224  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme bir `Is` yan tümcesi aşağıdaki `TypeOf` işleci; Örneğin, `TypeOf MyControl Is ComboBox`.
