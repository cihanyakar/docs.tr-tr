# <a name="shared-sub-new-cannot-have-any-parameters"></a>Paylaşılan 'Sub New' parametreleri olamaz
Bir parametre veya parametreler bir paylaşılan dahil `Sub New` deyimi geçerli değil.  
  
 **Hata Kimliği:** BC30479  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Parametre veya parametreler kaldırın.
