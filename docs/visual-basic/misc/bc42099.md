# <a name="unused-local-constant-ltconstantgt"></a>Kullanılmayan yerel sabit: '&lt;sabit&gt;'
Bir yerel sabit herhangi bir yerde kodunuzda kullanılmıyor bildirilmiş.  
  
 **Hata Kimliği:** BC42099  
  
### <a name="to-remove-this-warning"></a>Bu uyarıyı kaldırmak için  
  
1.  Bu hata, bir uyarıdır ve yok sayılabilir. Bu uyarıyı kaldırmak için kullanılmayan Sabit bildiriminde kodunuzdan kaldırın.
