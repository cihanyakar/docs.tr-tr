# <a name="ltmembergt-implicitly-defined-for-lteventnamegt-cannot-shadow-a-mustoverride-method-in-the-base-ltclassgt-ltclassnamegt"></a>'&lt;üye&gt;', için örtük tanımlanırsa'&lt;eventname&gt;', bir 'MustOverride' yöntemini gölgeleyemez &lt;sınıfı&gt; '&lt;classname&gt;'
Bir yöntem ile bildirilen şekilde belirtilen olay aynı ada sahip bir üye örtük olarak bildiriyor. `MustOverride` değiştiricisi.  
  
 **Hata Kimliği:** BC31413  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `MustOverride` değiştirici yöntemi temel sınıf veya özellik verin ya da yöntem benzersiz bir ad.
