# <a name="variables-in-modules-cannot-be-declared-ltspecifiergt"></a>Modül içindeki değişkenler bildirilemez '&lt;belirticisi&gt;'
Gibi bir belirleyici kullanmış `MustInherit`, içinde bir değişken üzerinde bir `Module` deyimi. Modüller asla oluşturulamaz, devralmayı desteklemeyen ve arabirimler uygulanamaz.  
  
 **Hata Kimliği:** BC30593  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcısını kaldırın.
