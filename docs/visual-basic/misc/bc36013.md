# <a name="on-error-statements-are-not-valid-within-using-statements"></a>'On Error' deyimleri 'Using' deyimleri içinde geçerli değildir
Bir `On Error` deyimi görünür içinde bir `Using` deyimi ancak bu bağlamda geçerli değil.  
  
 **Hata Kimliği:** BC36013  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yapılandırılmış hata işleme, aşağıdaki gibi kullanan bir `Try…Catch` bloğunda yerini `On Error` deyimi.
