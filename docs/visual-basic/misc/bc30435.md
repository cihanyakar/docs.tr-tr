# <a name="members-in-a-structure-cannot-be-declared-ltspecifiergt"></a>Yapıdaki üyeler bildirilemez '&lt;belirticisi&gt;'
İçinde bir üyesi üzerinde geçerli değil bir tanımlayıcı kullandığınız bir `Structure` deyimi.  
  
 **Hata Kimliği:** BC30435  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcısını kaldırın.
