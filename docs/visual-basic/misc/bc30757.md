# <a name="goto-ltlabelnamegt-is-not-valid-because-ltlabelnamegt-is-inside-a-for-or-for-each-statement-that-does-not-contain-this-statement"></a>' GoTo &lt;labelname&gt;' geçerli değil çünkü '&lt;labelname&gt;' içinde bir 'For' veya 'For Each' bu deyimi içermeyen deyimi
`GoTo` Geçerli kod bloğunun içinde atlama deyimleri kısıtlanır.  
  
 **Hata Kimliği:** BC30757  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kodunuzu yeniden yapılandırabilirsiniz. böylece `GoTo` deyimi ve etiketi olan içinde hem de `For` blok.
