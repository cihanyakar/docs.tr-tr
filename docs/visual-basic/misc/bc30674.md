# <a name="end-synclock-must-be-preceded-by-a-matching-synclock"></a>'SyncLock End', eşleşen bir 'SyncLock tarafından' gelmelidir
`SyncLock` ile başlayan blokları `SyncLock` anahtar sözcüğü ve sonu `End SyncLock` oluşturun.  
  
 **Hata Kimliği:** BC30674  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Emin olun `SyncLock` blok ile başlayan bir `SyncLock` deyimi.
