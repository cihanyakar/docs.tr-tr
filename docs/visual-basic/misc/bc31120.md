# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>'Son RaiseEvent' satırdaki ilk deyim olmalıdır
Bir `End RaiseEvent` deyimi bir iki nokta üst üste (:) deyim ayırıcısı izler. `End RaiseEvent` Kaynak satırında tek deyim olması gerekir.  
  
 **Hata Kimliği:** BC31120  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok deyime farklı satırlarına ayırın.
