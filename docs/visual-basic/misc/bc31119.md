# <a name="end-removehandler-must-be-the-first-statement-on-a-line"></a>'End RemoveHandler' satırdaki ilk deyim olmalıdır
Bir `End RemoveHandler` deyimi bir iki nokta üst üste (:) deyim ayırıcısı izler. `End RemoveHandler` Kaynak satırında tek deyim olması gerekir.  
  
 **Hata Kimliği:** BC31119  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok deyime farklı satırlarına ayırın.
