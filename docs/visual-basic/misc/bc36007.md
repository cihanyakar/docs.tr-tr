# <a name="end-using-must-be-preceded-by-a-matching-using"></a>'End Using' bir eşleşen 'Using tarafından' gelmelidir
Bir `End Using` deyimi görünür eşleşme ile `Using` kendisinden bildirimi.  
  
 **Hata Kimliği:** BC36007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `End Using` yedekli ise deyimi.  
  
-   Eksik olan kaynağı [Using deyimi](../../visual-basic/language-reference/statements/using-statement.md) eksik ise.  
  
-   Taşıma `End Using` kod içinde uygun konuma deyimi.
