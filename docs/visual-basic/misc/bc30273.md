# <a name="ltspecifiergt-is-not-valid-on-an-interface-property-declaration"></a>'&lt;belirticisi&gt;' arabirim özelliği bildiriminde geçerli değil
A `Property` deyimi içinde bir arabirim içeren bir geçersiz anahtar sözcüğü gibi `Implements`. Bir arabirim, üyeleri yalnızca tanımlayın, uygulamadan değil.  
  
 **Hata Kimliği:** BC30273  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz anahtar sözcük bildirimi deyimden kaldırın.  
  
-   Arabirim üyelerini uygulaması arabirimi uygulayan bir sınıfa taşıyın.
