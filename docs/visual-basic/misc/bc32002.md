# <a name="ltkeywordgt-is-valid-only-within-a-class"></a>'&lt;anahtar sözcüğü&gt;' yalnızca bir sınıf içinde geçerlidir
Bir anahtar sözcük ilgili sınıflar için gibi `Me` veya `MyClass`, bir sınıf tanımının dışında kullanılır.  
  
 **Hata Kimliği:** BC32002  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Anahtar sözcüğü kullanılarak kodu sınıf örneklerinin içeriyorsa, bu dosyayı bir sınıf uygulamasına başına taşıyın.  
  
-   Anahtar sözcüğü kullanılarak kod sınıflar için geçerli değilse, geçersiz anahtar sözcük kaldırın.
