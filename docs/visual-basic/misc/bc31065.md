# <a name="set-parameter-cannot-be-declared-ltspecifiergt"></a>'Set' parametresi olarak bildirilemez '&lt;belirticisi&gt;'
Geçersiz bir Değiştirici bir parametre ile belirtilen bir `Set` özellik yordamı.  
  
 **Hata Kimliği:** BC31065  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz değiştirici gibi kaldırmak `ByRef`, `ParamArray`, ve `Optional`.
