# <a name="argument-ltargumentnamegt-must-be-greater-than-or-equal-to-1"></a>Bağımsız değişken '&lt;argumentname&gt;' değerinden büyük veya 1'e eşit olmalıdır
Bir bağımsız değişkenin, 1'den küçük değerdir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bağımsız değişkenin değeri geçerli aralıkta olduğundan emin olun.  
  
2.  Bağımsız değişken bir hesaplamanın sağlanırsa, hesaplama işlemleri değerleri kontrol edin.
