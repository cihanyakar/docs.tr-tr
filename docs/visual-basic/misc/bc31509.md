# <a name="unable-to-open-resource-file-ltfilenamegt-lterrorgt"></a>Kaynak dosyası açılamıyor. '&lt;filename&gt;': &lt;hata&gt;
Belirtilen kaynak dosyası açılırken bir hata oluştu.  
  
 **Hata Kimliği:** BC31509  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Belirtilen ad ve konum doğru olduğundan emin olun.  
  
2.  Dosya zarar görmüş veya bozuk olmadığını doğrulayın.
