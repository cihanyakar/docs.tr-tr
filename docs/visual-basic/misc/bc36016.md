# <a name="implicit-variable-ltvariablenamegt-is-invalid-because-of-ltmessagegt"></a>Örtük değişkeni '&lt;variablename&gt;'nedeniyle geçersiz'&lt;ileti&gt;'
Geçersiz bir örtük değişken, hata ayıklama sırasında oluşturuldu.  
  
 **Hata Kimliği:** BC36016  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Nedeni ve olası çözümü için alıntı iletisini inceleyin.  
  
2.  Sorun devam ederse, koşullar hakkında bilgi toplamak ve Microsoft Ürün Destek Hizmetleri bilgilendirir.
