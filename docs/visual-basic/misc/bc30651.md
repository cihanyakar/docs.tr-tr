# <a name="ltpropertygt-parameters-cannot-be-declared-byref"></a>&lt;Özellik&gt; parametreleri 'ByRef' olarak bildirilemez
Özellikler için parametreleri kullanılarak bildirilmelidir `ByVal` değiştiricisi.  
  
 **Hata Kimliği:** BC30651  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Özellik parametreleri olarak tanımlayan `ByVal`.
