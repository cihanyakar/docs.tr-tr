# <a name="goto-statements-are-not-valid-in-the-immediate-window"></a>'GoTo' deyimleri yürütme penceresinde geçerli değildir
`GoTo` Deyimi dallanma gerçekleştirir ve hata ayıklama bir bağlamda izin verilmiyor.  
  
 **Hata Kimliği:** BC30133  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sorun değil bir `GoTo` deyiminde **hemen** penceresi.
