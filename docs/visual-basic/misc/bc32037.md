<span data-ttu-id="c8895-103">İç içe döngüleri, tek bir sonlandırılır `Next` daha fazla loop değişkenleri aşağıdaki örnekteki gibi iç içe döngüleri kullandığınızdan ifadesi:</span><span class="sxs-lookup"><span data-stu-id="c8895-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>

```  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```
