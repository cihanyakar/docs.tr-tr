# <a name="stop-statements-are-not-valid-in-the-immediate-window"></a>'Stop' deyimleri yürütme penceresinde geçerli değildir
`Stop` Ve `End` deyimleri yürütmeyi askıya almak ve bir hata ayıklama bağlamında izin verilmez.  
  
 **Hata Kimliği:** BC30122  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sorun değil bir `Stop` veya `End` deyiminde **hemen** penceresi.
